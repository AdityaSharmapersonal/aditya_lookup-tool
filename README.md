# Country Lookup

Country search app with structured facts first and a grounded AI explanation second.

## Stack

- Next.js App Router
- TypeScript
- Prisma
- PostgreSQL
- OpenAI structured outputs

## Environment

Create a `.env` file from `.env.example`.

## Scripts

- `npm install`
- `npx prisma generate`
- `npm test`
- `npm run build`
- `npm run dev`
